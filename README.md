# React Input Factory

This project shows how to implement simple input factory with React.

## Get started

1. `npm install`
2. `npm run dev`

## Project overview

| Folder     | Description                                                  |
| ---------- | ------------------------------------------------------------ |
| components | React Functional Components that will be used in the project |
| types      | Custom RFC types                                             |
| pages      | Folder with web pages                                        |
